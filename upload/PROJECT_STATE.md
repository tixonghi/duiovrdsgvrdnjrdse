## 🔜 Запланированные задачи (на завтра)

### Персонализированная разминка

**Требования:**
1. Создать базу разминочных упражнений с категориями:
   - Кардио (jumping jacks, high knees, burpees)
   - Суставная гимнастика (вращения, махи)
   - Динамическая растяжка (выпады с поворотом, проходка)

2. Логика генерации по цели:
   - **fat_loss:** упор на кардио (поднять пульс)
   - **muscle_gain:** суставы + разогрев мышц
   - **endurance:** кардио с прогрессией
   - **maintenance:** сбалансированный подход

3. Персонализация по физическим параметрам:
   - Высокий вес → исключить прыжки
   - Пожилой возраст → плавные движения
   - Травмы → учитывать ограничения

4. UI интеграция:
   - Блок разминки перед основными упражнениями
   - Таймер для упражнений на время
   - Прогресс выполнения разминки
